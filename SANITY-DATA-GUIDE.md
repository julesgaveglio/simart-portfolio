## 6. Instructions pour créer un document dans Sanity Studio

1. Accédez à Sanity Studio à l'adresse http://localhost:3333
2. Cliquez sur le type de document que vous souhaitez créer (Series, Work ou About)
3. Cliquez sur "Create new document"
4. Remplissez les champs selon les informations fournies ci-dessus
5. Pour les images, vous pouvez télécharger une image grise temporaire ou utiliser l'option "Upload" pour ajouter vos propres images
6. Pour les œuvres, n'oubliez pas de sélectionner la série correspondante dans le champ "Series"
7. Cliquez sur "Publish" pour enregistrer le document

## 7. Synchronisation avec le site

Une fois que vous avez créé tous les documents dans Sanity Studio, vous pouvez synchroniser les données avec votre site en visitant :

```
http://localhost:3000/api/revalidate-all
```

Cette URL forcera Next.js à récupérer les données les plus récentes de Sanity et à mettre à jour votre site.
